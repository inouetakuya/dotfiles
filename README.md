# My dotfiles

HOME 配下の dotfiles を管理するリポジトリ。

## Install

```shell
cd ~
git clone git@github.com:inouetakuya/dotfiles.git
```

## Setup

```shell
sh ~/dotfiles/bin/init.sh
bundle install
```
